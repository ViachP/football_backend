FlashScore.com получает данные из нескольких источников:

1. Sportradar - один из крупнейших поставщиков спортивных данных в реальном времени
2. Stats Perform (ранее Opta Sports) - предоставляет детальную статистику и аналитику
3. IMG Arena - live данные и коэффициенты букмекеров
4. Genius Sports - официальные данные лиг и федераций
5. Прямые партнерства с лигами (Premier League, NBA, UEFA и др.)

Для вашего приложения я использовал моковые данные (тестовые данные), которые имитируют структуру реальных API. В файлах mocks/matches.ts и mocks/standings.ts созданы примеры данных с:

- Результатами матчей
- Коэффициентами букмекеров  
- Турнирными таблицами
- Статистикой команд

Если вы хотите использовать реальные данные, можно подключить:
- API-Football (бесплатный тариф до 100 запросов/день)
- TheSportsDB (бесплатный для некоммерческого использования)
- Sportradar API (платный, но очень качественный)
